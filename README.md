# UpDownGame
0 부터 30 사이의 랜덤 숫자를 맞추는 게임

## How it works
Slieder를 통해 숫자를 정하고 Hit Button 을 통해 답을 제출한다.<br/>
5번의 제출 기회가 있으며, 기회 소진 시 게임은 리셋된다.<br/>
또한 5번의 기회 이전에 정답을 맞출 시, 게임은 리셋된다.<br/>
